<p align="center">
	<b>fullpath(path) - get fullpath</b>
</p>

#### Install

pip: 
`[sudo] pip install fullpath`

#### Usage

```python
	from fullpath import *

	>>> fullpath("~")
	'/Users/username'

	>>> fullpath(".")
	'/path/to/current/directory'

	>>> fullpath("..")
	'/path/to/current'
```
