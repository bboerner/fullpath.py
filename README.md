#### Install

`[sudo] pip install fullpath`

#### Usage

```python
>>> from fullpath import fullpath

>>> fullpath("~")
```

#### Example

```python
>>> fullpath("~")
'/Users/username'

>>> fullpath(".")
'/path/to/current/directory'

>>> fullpath("..")
'/path/to/current'
```
